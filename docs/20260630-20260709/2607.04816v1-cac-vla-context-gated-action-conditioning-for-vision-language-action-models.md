## 摘要
视觉-语言-动作（VLA）模型已成为通用机器人操作的一种有前景的范式，其中视觉-语言表示用于条件化连续动作生成。然而，这些表示并未针对动作条件化进行显式优化，使得动作专家需要弥合多模态理解与精确运动控制之间的差距。最近的动作推理方法引入了额外模块来生成显式动作计划或动作空间推理信号，证明了动作级指导的益处，但通常需要独立的动作生成框架。我们提出CAC-VLA，一种上下文门控动作条件化框架，直接在VLM内部学习轻量级潜在动作接口。CAC-VLA不生成可执行轨迹，而是训练VLM预测从粗到细的潜在动作（这些是从未来动作片段编码的结构化表示），并通过上下文门自适应地利用它们来条件化动作专家。这使得VLM原生的动作条件化成为可能，同时校准潜在动作指导对专家动作生成的影响。在LIBERO和LIBERO-Plus上的实验证明了CAC-VLA的有效性，在LIBERO上平均成功率达到98.3%，在LIBERO-Plus上达到89.5%，表明上下文门控潜在动作条件化是连续专家控制的有效接口。

## Abstract
Vision-Language-Action (VLA) models have become a promising paradigm for generalist robot manipulation, where visual-language representations are used to condition continuous action generation. However, these representations are not explicitly optimized for action conditioning, leaving the action expert to bridge the gap between multimodal understanding and precise motor control. Recent action-reasoning methods introduce additional modules to generate explicit action plans or action-space reasoning signals, demonstrating the benefit of action-level guidance but often requiring separate action-generation frameworks. We propose CAC-VLA, a Context-Gated Action Conditioning framework that learns a lightweight latent-action interface directly within the VLM. Instead of generating executable trajectories, CAC-VLA trains the VLM to predict coarse-to-fine latent actions, which are structured representations encoded from future action segments, and adaptively leverages them to condition the action expert via a context gate. This enables VLM-native action conditioning while calibrating the influence of latent-action guidance on expert action generation. Experiments on LIBERO and LIBERO-Plus demonstrate the effectiveness of CAC-VLA, achieving 98.3% average success rate on LIBERO and 89.5% LIBERO-Plus, suggesting that context-gated latent-action conditioning is an effective interface for continuous expert control.

## 2. 方法论：核心技术细节
### 2.1 核心思想
- 在VLA框架中引入**VLM原生的潜在动作接口**：VLM通过特定查询令牌（query tokens）预测**潜在动作**，这些潜在动作是未来动作片段的结构化编码（来自顺序动作分词器OAT），但**不直接作为可执行动作**，而是作为动作专家的结构化条件信号。
- 通过**上下文门控机制**自适应地调节潜在动作指导对动作专家的注入强度，避免固定强度带来的僵化影响。

### 2.2 技术细节
1. **潜在动作预测**：
   - 使用OAT（Ordered Action Tokenizer，冻结）将未来动作片段 \(a_{t:t+H_l-1}\) 编码为原始潜在向量 \(z_t^{oat}\)（维度 \(D_z=4\)，潜在动作时间跨度 \(H_l\) 可配置）。
   - 在VLM输入中附加可学习的潜在查询令牌 \(q\)（数量 \(N_q=8\)），使用专家感知注意力掩码（如图1(b)），避免查询令牌直接与动作专家交互，确保VLM输出查询隐藏状态 \(h_q^t\) 后，通过轻量投影头 \(\hat{z}_t = W_z \text{LN}(h_q^t)\) 预测潜在动作。
   - 对齐损失：使用令牌级Smooth-L1损失 \(\mathcal{L}_{\text{align}}\) 监督预测 \(\hat{z}_t\) 与编码目标 \(z_t^{oat}\) 的差异。

2. **上下文门控动作条件化**：
   - 训练时，使用编码目标 \(z_t^{oat}\)；推理时，使用VLM预测的 \(\hat{z}_t\) 作为条件源。
   - 通过线性投影 \(\phi_c\) 将潜在动作映射为专家兼容的条件令牌 \(c_t\)。
   - **跨模态交叉注意力**：在动作专家每个Transformer层中，归一化后的动作隐藏状态 \(\bar{x}_l\) 作为Query，条件令牌 \(c_t\) 作为Key和Value，检索潜在动作更新 \(u_l\)。
   - **上下文门控**：对归一化动作状态和检索更新分别进行平均池化，通过两层MLP（tanh + sigmoid）生成通道级门控值 \(g_l \in \mathbb{R}^{B \times 1 \times D}\)，最终通过门控残差连接 \(x_l^+ = x_l + g_l \odot u_l\) 注入更新。该方式分离了“检索”与“注入”，门控值自适应于当前动作生成状态。

3. **训练与推理**：
   - 总体损失：\(\mathcal{L} = \mathcal{L}_{\text{act}} + \lambda_{\text{align}} \mathcal{L}_{\text{align}}\)，其中 \(\mathcal{L}_{\text{act}}\) 为原始流匹配动作专家损失，\(\lambda_{\text{align}} = 0.1\)。
   - 推理阶段移除OAT分词器，仅依赖VLM预测的潜在动作；训练时使用未来动作片段作为监督，保证VLM原生部署。
