## 摘要
视觉-语言-动作（VLA）模型常在环境变化下（如相机位姿改变或切换到不同但类似的机器人，例如从Panda到UR5e）无法执行已学会的任务。将这些模型适配到变化后的环境（即目标域）通常需要每个任务收集多个演示，成本高昂。为减轻数据收集和训练的负担，我们提出一种基于类比的方法，通过权重向量算术和添加域特定信息来适配环境变化下的VLA模型，命名为域算术（DART）。与先前方法不同，DART只需收集单个演示，实现高效适配。为精确隔离要添加的域特定信息，DART在权重向量的奇异分量之间进行子空间对齐，以滤除噪声分量。在模拟和真实世界实验中，DART在跨多种视觉和本体变化的单次场景下优于现有VLA适配方法。代码见https://github.com/snumprlab/dart。

## Abstract
Vision-Language-Action (VLA) models often fail to perform the same learned tasks under environmental shifts, such as changes in camera pose and shifts to a different but similar robot (e.g., from Panda to UR5e). Adapting these models to the shifted environment (i.e., target domain) often requires training on multiple demonstrations for each task, which are costly to collect. To reduce the burden of data curation and training, we propose an analogy-based method that adapts VLA models under environmental shifts through weight vector arithmetic with domain-specific information addition, named Domain ARiThmetic (DART). Unlike prior approaches, DART requires collecting only a single demonstration, enabling efficient adaptation. To accurately isolate domain-specific information for addition, DART performs subspace alignment between singular components in weight vectors to filter out noisy components. In both simulated and real-world experiments, DART outperforms existing VLA adaptation methods in one-shot scenarios across diverse visual and embodiment shifts. Code is available at https://github.com/snumprlab/dart.

## 一、论文的核心问题与整体含义

- **研究动机**：视觉-语言-动作（VLA）模型在训练环境（源域）中表现优异，但在部署时遇到环境变化（如相机视角改变、传感器标定差异、机器人本体更换）会导致性能大幅下降。现有适配方法通常需要大量目标域演示数据，收集成本高昂，尤其不适合家庭等场景。
- **研究目标**：实现**单次（one-shot）VLA适配**——仅需利用目标域的**一个任务的一个演示**，即可将源域训练的多任务VLA策略迁移到新环境，同时保持原有多任务能力。
- **核心挑战**：单次微调的权重更新主要包含任务特定方向，缺乏跨任务可迁移的域知识；直接微调会破坏源域能力。

## 二、论文提出的方法论：DART（Domain ARiThmetic）

### 核心思想
基于权重算术中的**类比操作**（analogy operation），将域知识表示为可加性权重向量。受“queen = king + woman - man”启发，通过**源域和目标域更新向量的减法**来提取纯域向量，再将其加回基模型。

### 关键技术细节

1. **更新向量定义**：对同一任务 \(T_m\)，分别在源域和目标域收集单次演示，微调得到参数 \(\theta_{m,\text{src}}\) 和 \(\theta_{m,\text{tgt}}\)，定义更新向量：
   \[
   \Delta_{m,\text{src}} = \theta_{m,\text{src}} - \theta_0, \quad \Delta_{m,\text{tgt}} = \theta_{m,\text{tgt}} - \theta_0
   \]

2. **域向量提取（类比操作）**：
   \[
   \delta_{\text{tgt}} = \Delta_{m,\text{tgt}} - \Delta_{m,\text{src}}
   \]
   通过相减消除共享的任务方向，保留域特定方向。

3. **子空间滤波（Subspace Filtering）**：直接相减会引入源域噪声和微调噪声。方法：
   - 对 \(\Delta_{m,\text{src}}\) 和 \(\Delta_{m,\text{tgt}}\) 进行奇异值分解（SVD）。
   - 计算交互矩阵 \(C = U_{\text{tgt}}^\top U_{\text{src}}\)，得到每个源基向量在目标子空间中的重叠能量 \(e_j = \|C_{:,j}\|_2^2\)。
   - 基于子空间对齐分数 \(\gamma\) 动态选择能量阈值，保留与目标子空间高度对齐的源基向量，滤除噪声分量。
   - 加权域向量：\(\tilde{\delta}_{\text{tgt}}^{(l)} = \gamma^{(l)} \cdot (\Delta_{m,\text{tgt}}^{(l)} - \tilde{\Delta}_{m,\text{src}}^{(l)})\)。

4. **域向量注入**：
   \[
   \theta^* = \theta_0 + \alpha \cdot \tilde{\delta}_{\text{tgt}}
   \]
   其中 \(\alpha\) 为缩放系数。

### 算法流程（文字描述）
- 输入：基策略参数 \(\theta_0\)，适应任务 \(T_m\)，源域数据 \(D_{m,\text{src}}\)，目标域数据 \(D_{m,\text{tgt}}\)，系数 \(\alpha\)。
- 步骤：
  1. 分别对源域和目标域单次演示进行微调（1000步），得到 \(\theta_{m,\text{src}}, \theta_{m,\text{tgt}}\)。
  2. 计算更新向量 \(\Delta_{m,\text{src}}, \Delta_{m,\text{tgt}}\)。
  3. 对每个二维层进行SVD，计算子空间对齐分数 \(\gamma\) 和重叠能量，进行子空间滤波与缩放。
  4. 将精炼域向量 \(\tilde{\delta}_{\text{tgt}}\) 缩放后加回 \(\theta_0\)。
- 输出：适配后的多任务策略 \(\theta^*\)。

## 六、论文的主要结论与发现

1. **单次微调权重可近似分解为可加性的任务和域方向**：通过子空间对齐实验验证，域方向在权重空间中具有结构化和可组合性。
2. **DART显著优于现有方法**：在LIBERO、MimicGen和真实世界实验中，DART在所有视角和扰动下平均提升约10-30个百分点。
3. **域向量可合并**：将多个目标域的域向量通过模型合并方法整合，单个向量可适配所有域，减少存储开销。
4. **对多种VLA架构普适**：在π0.5和π0-FAST上均有效。
5. **适应后源域性能保持良好**：在源域上仅小幅下降（约2-4%），优于微调导致的灾难性遗忘。

## 七、优点

1. **数据高效**：仅需一个目标域演示和一个源域演示（均已知），显著降低部署数据成本。
2. **架构无关**：无需修改模型结构，适用于多种VLA骨干。
3. **理论启发性**：揭示并利用了权重空间中任务与域方向的可加性，提供深层理解。
4. **子空间对齐增强稳健性**：通过奇异值分解和能量过滤，有效抑制微调噪声，提升域向量提取质量。
