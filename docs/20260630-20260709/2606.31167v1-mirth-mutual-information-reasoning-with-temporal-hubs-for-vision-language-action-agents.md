## 摘要
VLA模型已作为一种强大的范式出现，用于将网络规模数据的语义知识迁移到物理机器人控制中。然而，当前的单帧架构存在固有限制：时间短视导致丢弃历史动态，高级指令与低级运动命令之间的推理鸿沟，以及自回归标量解码导致的推理效率低下。在这项工作中，我们提出了MIRTH，一个旨在应对这些挑战的统一框架。MIRTH通过三项关键创新增强了预训练的VLA骨干网络：（1）双尺度时间记忆枢纽，将长期场景演变和短期运动趋势压缩为紧凑嵌入；（2）通过互信息目标优化的潜在推理令牌，刻画出语义计划空间以对齐多模态上下文与动作轨迹；（3）一种并行动作解码方案，用向量级预测替代自回归生成以最大化控制吞吐量。在LIBERO仿真基准和真实世界LeRobot平台上的广泛评估表明，MIRTH达到了最先进的性能，并展现出新兴的错误恢复能力。代码和收集的数据集已在http://github.com/kiva12138/mirth发布。

## Abstract
VLA models have emerged as a powerful paradigm for transferring semantic knowledge from web-scale data to physical robotic control. However, current single-frame architectures suffer from intrinsic limitations: temporal myopia that discards historical dynamics, reasoning gaps between high-level instructions and low-level motor commands, and inference inefficiency due to autoregressive scalar decoding. In this work, we propose MIRTH, a unified framework designed to address these challenges. MIRTH augments a pretrained VLA backbone with three key innovations: (1) dual-scale temporal memory hubs that compress long-term scene evolution and short-term motion trends into compact embeddings; (2) latent reasoning tokens optimized via a mutual-information objective carving out a semantic plan space to align multimodal context with action trajectories; and (3) a parallel action decoding scheme that replaces autoregressive generation with vector-wise prediction to maximize control throughput. Extensive evaluations on the LIBERO simulation benchmark and a real-world LeRobot platform demonstrate that MIRTH achieves state-of-the-art performance and exhibiting emergent error recovery capabilities. The codes and collected datasets are released at http://github.com/kiva12138/mirth.

## 1. 核心问题与研究动机

当前视觉-语言-动作（VLA）模型（如OpenVLA）主要采用**单帧架构**，存在三大固有局限：
- **时间短视**：仅依赖当前观测，丢弃历史动态，导致物体被遮挡后无法跟踪。
- **推理鸿沟**：高级语言指令与低级动作命令之间缺乏可解释的中间规划，直接映射或离散动作令牌均存在不足。
- **推理效率低**：自回归标量解码（将连续动作量化为离散令牌）导致高延迟，难以满足实时控制。

论文旨在统一解决上述问题，提出**MIRTH**框架，在保持VLM语义丰富性的同时实现历史感知、可推理、高效率的机器人控制。

## 2. 方法论：核心思想与关键技术

### 整体架构
MIRTH基于预训练VLA骨干（OpenVLA），引入三个创新模块：

#### (1) 双尺度时间记忆中枢（Temporal Memory Hubs）
- **长时工作记忆（Workspace Memory Hub）**：利用K个不同衰减率的指数移动平均（EMA）压缩长期场景布局，通过MLP预测每个patch的混合权重，自适应地集成慢变上下文和快动动态。
- **短时运动记忆（Short-Horizon Memory Hub）**：维护最近w帧的固定长度队列，通过带温度与近因偏置的时间注意力聚合局部运动信息。
- **融合**：使用sigmoid门控将两个记忆输出融合，并通过**前缀（Prefix）**或**注入（Infusion）**方式整合到当前帧表示中。前缀方式将记忆展平后添加到序列前，效果更好（但效率略低）。

#### (2) 基于互信息的潜在推理令牌（Latent Reasoning Tokens）
- 在输入序列中插入可学习的推理令牌（位于上下文令牌与动作令牌之间）。
- 通过**InfoNCE对比损失**最大化推理令牌与上下文、推理令牌与动作轨迹之间的互信息，无需文本标注即可形成语义规划空间。
- 损失函数：\( \mathcal{L}_{mi} = \lambda_{ra}\mathcal{L}_{ra} + \lambda_{rx}\mathcal{L}_{rx} \)，分别对齐推理-动作和推理-上下文。

#### (3) 并行动作解码（Parallel Action Decoding）
- 每个时间步用一个动作令牌表示完整动作向量（而非每个自由度一个令牌），隐藏状态通过轻量级投影头并行映射为连续动作。
- 训练采用\( \ell_1 \)回归损失，结合互信息损失共同优化。

### 公式与流程（文字说明）
- 输入序列：`[记忆令牌；推理令牌；语言指令；视觉/本体感知令牌；动作令牌]`，使用全因果注意力。
- 输出：一次性生成动作块（chunk），大小默认为10步。

## 6. 主要结论与发现

- MIRTH在LIBERO上达到平均98.1%成功率，显著超越OpenVLA（76.5%）等基线，尤其在长时任务（LIBERO-Long 95.3% vs 53.7%）。
- LeRobot真实实验中，MIRTH在复杂推理任务上拉开大幅差距，并保持高吞吐量（约62-70Hz）。
- **核心发现**：
  - 双尺度记忆有效捕获运动动态（线性探针误差远低于单帧）。
  - 帧顺序扰乱导致成功率下降7.3%，证明模型依赖因果时序。
  - MI约束下的推理令牌自发形成任务语义簇（t-SNE可视化），无需文本标注。
  - MIRTH展现出**12.1%的错误恢复率**，而基线仅5.2%以下，表明潜在推理支持重规划。

## 7. 优点

1. **统一解决三大关键局限**：时间记忆、潜在推理、高效解码，设计完整。
2. **记忆紧凑高效**：固定长度提示/注入，不随历史增长而增加计算，推理吞吐量高。
3. **潜在推理免标注**：互信息损失替代昂贵的文本中间监督，且能自然聚类任务语义。
4. **并行动作解码**：将自回归开销从动作维度转移到单次投影，显著提升控制频率。
5. **开源完整代码与数据集**（GitHub链接），促进复现与扩展。
6. **全面实验验证**：覆盖模拟和真实，消融充分，分析深入（线性探针、t-SNE、恢复率等）。

## 8. 不足与局限

1. **可解释性差**（作者明确提及）：潜在推理令牌不生成人类可读文本，调试和审计困难。
2. **应用范围有限**：仅验证了固定单臂操作场景，未涉及双臂、移动机器人或人形机器人。
3. **长时记忆容量上限**：压缩的记忆（固定长度）可能丢失极长任务（数百步）的早期关键信息，存在遗忘风险。
4. **潜在社会偏见**：继承自预训练VLM（可能含网络数据偏见），虽聚焦操作但仍有残留风险。
5. **安全风险**：并行动作解码虽快，但模型在分布外场景可能产生不可预测行为，需硬件急停机制保护。
6. **实验覆盖不足**：仅在一个真实平台（LeRobot）进行测试，未在其他通用机器人平台（如Franka、xArm）上验证泛化性。
